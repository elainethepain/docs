## Prerequisites

- Node.js 18+ installed
- Basic knowledge of Next.js
- Kamino API key (optional for public endpoints)

## Quick Setup

```bash
npx create-next-app@latest kamino-example
cd kamino-example
npm install @kamino-finance/sdk
```

## Example Code

```javascript
// pages/api/kamino.js
export default async function handler(req, res) {
  const response = await fetch('https://api.kamino.finance/staking-yields/tokens/mSoLzYCxHdYgdzU16g5QSh3i5K3z3KZK7ytfqcJm7So/history?start=2024-01-01T00:00Z&end=2024-02-01T01:00Z');
  const data = await response.json();
  
  res.status(200).json(data);
}
```
